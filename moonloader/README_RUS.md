## TrailFX lua скрипт
Горячая клавиша запуска/деактивации скрипта: `G`. Можно изменить в коде скрипта, в функции `OnUpdateKeys()`:
```lua
function OnUpdateKeys()
	if isKeyJustPressed(keys.VK_G) then -- Менять здесь. Например для клавиши 'H' нужно VK_G заменить на VK_H
		TrailFXCommand()
	end
end
```
Основная команда скрипта: `/trailfx`. Если вводить без указания `id` игрока, то активирует/деактивирует скрипт. Если ввести `/trailfx [id]`, то добавит треил другому игроку (игрок должен находиться рядом с вами и быть пилотом в транспорте, который поддерживается скриптом).

> [!NOTE] 
> Данный функционал ещё не был достаточно протестирован, возможны сбои в работе при добавлении нескольких игроков

### Добававление новых самолётов, редактирование существующих
Основная информация о всех поддерживаемых самолётах хранится в таблице `dbOffset`. Данная таблица заполняется каждый раз при инициализации скрипта, в функции `InitDB()`.

Каждая запись в таблице имеет следующий вид:

```lua
dbOffset 
{ 
    particleName1 = string "название эффекта 1 (FXSystem)", 
    particleName2 = string "название эффекта 2 (FXSystem)", 
    modelID = int модель_транспорта, 
    x1 = float координата_X_эффекта_1, 
    y1 = float координата_Y_эффекта_1, 
    z1 = float координата_Z_эффекта_1, 
    x2 = float координата_X_эффекта_2, 
    y2 = float координата_Y_эффекта_2, 
    z2 = float координата_Z_эффекта_2 
}
```

Для добавление нового самолёта в таблицу, добавьте в функции `InitDB()` запись вида:

```lua
table.insert(dbOffset, { particleName1 = "trail_white_long", particleName2 = "trail_blue_long", modelID = 513, x1 = 3, y1 = -0.2, z1 = -1.5, x2 = -3, y2 = -0.2, z2 = -1.5 })
```

и в функцию `GetPlaneOffsetType(modelID)` добавьте новую запись вида `elseif modelID == int id_модели_самолёта then return int следующий_номер_по_порядку`, например:

```lua
function GetPlaneOffsetType(modelID)
	if modelID == 520 then return 1
	elseif modelID == 476 then return 2
    -- Добавлена новая запись ниже
	elseif modelID == 513 then return 3 -- в return указано следующее число по порядку 
	end
	return nil
end
```
